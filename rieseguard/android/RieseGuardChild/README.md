# RieseGuardChild Android Application

This folder is the designated directory for the RieseGuard child-side Android application.

## Getting Started

To initialize a new Android project here:
1. Open Android Studio.
2. Select **New Project** -> **Empty Activity** (or another template).
3. Set the project location to this folder: `rieseguard/android/RieseGuardChild/`.
4. Configure package name (e.g., `com.rieseguard.child`), Minimum SDK (e.g., API 26 or higher), and build configuration language (Kotlin DSL recommended).
5. Click **Finish**.
